🔴 ACTIONS MANUELLES REQUISES (6 étapes, ~15 min)

  1️⃣  Créer la base D1 (2 min)
      → npx wrangler login
      → npx wrangler d1 create SPEEDV3
      → Copier le database_id dans wrangler.toml

  2️⃣  Créer le bucket R2 (2 min)
      → npx wrangler r2 bucket create SPEEDV3
      → Activer l'accès public sur le dashboard
      → Copier l'URL dans worker/index.js ligne 732

  3️⃣  Configurer les secrets (2 min)
      → npx wrangler secret put DEFAULT_ADMIN_USERNAME
      → npx wrangler secret put DEFAULT_ADMIN_PASSWORD

  4️⃣  Déployer le worker (2 min)
      → npx wrangler deploy
      → Noter l'URL et la mettre dans .env.local

  5️⃣  Initialiser la base (1 min)
      → curl https://SPEEDV3.VOTRE-USERNAME.workers.dev/api/init

  6️⃣  Déployer sur Vercel (5 min)
      → git push -u origin main
      → Importer sur Vercel
      → Configurer VITE_API_URL
